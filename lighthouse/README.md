# Lighthouse Reports

Store Lighthouse audit exports in this folder.

## Naming Convention

Use a timestamp and page context, for example:

- `home-mobile-2026-03-06.json`
- `home-mobile-2026-03-06.html`
- `contact-desktop-2026-03-06.json`

## What to Track

- Performance
- Accessibility
- Best Practices
- SEO
