# Visual Documentation

Add screenshots that prove key functionality and responsive behavior.

## Screenshot Storage Path

Place all images here:

- `docs/assets/screenshots/`

## Required Screenshots

- `home-desktop.png`
- `home-mobile.png`
- `services-tabs.png`
- `portfolio-filter.png`
- `contact-form-validation.png`
- `theme-dark-mode.png`

## How to Capture

1. Run `npm run dev`.
2. Open browser DevTools and switch device modes.
3. Capture desktop and mobile views.
4. Save images with the names above.
