# Project Overview

## Objective

Build a modern, multi-page React business website that demonstrates advanced frontend foundations:

- HTML5 semantics and accessible structure
- CSS Grid/Flexbox responsive design
- JavaScript ES6+ patterns
- React component architecture and hooks
- Performance-aware loading and routing

## Scope

Implemented pages:

1. Home
2. Services
3. Portfolio
4. About
5. Contact

Core user features:

- Responsive navigation with mobile menu
- Dark and light theme toggle
- Portfolio filtering and search
- Contact and newsletter forms with validation
- FAQ accordion and testimonial slider

## Success Criteria

- Fully responsive across mobile, tablet, and desktop
- WCAG-minded semantics and keyboard support
- Maintainable component and style architecture
- Passing lint/build checks in CI
